# Skills Index

Este indice clasifica los skills que conviene reutilizar en nuevos proyectos.

## Skills Base No-IA

| Skill                         | Cuando usar                                                   |
| ----------------------------- | ------------------------------------------------------------- |
| `base-fullstack-architecture` | Al crear o modernizar una app completa con esta arquitectura. |
| `hono-libsql-turso`           | Al tocar backend, rutas, DB, migraciones o Turso.             |
| `react-vite-operational-ui`   | Al crear UI React/Vite operacional.                           |
| `coolify-docker-minio`        | Al preparar Docker, Coolify, MinIO y readiness.               |
| `agent-workflow-quality`      | Al trabajar con agentes, reglas, quality gates y docs.        |
